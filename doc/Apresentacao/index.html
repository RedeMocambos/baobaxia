O que é o Baobáxia

É uma plataforma distribuída de comunicação que funciona como Rede com ou sem internet

A aplicação inicial do Baobáxia é uma Rede de Acervos digitais multimídia (áudio, vídeo, imagens e textos)

Origens

Referência ao Baobá, árvore que vive milhares de anos

O Baobá é simbolo da ancestralidade africana

O Baobá representa o lugar simbolico da memória nas comunidades

Baobáxia é a união de Baobá com Galáxia

Galáxia de memórias coletivas locais ligadas ao território

Mucua

A Mucua é o nome do fruto do Baobá

Mucua é o como chamamos os nós da rede Baobáxia.

Cada comunidade integrada ao Baobáxia possui uma Mucua, que é um computador com o software Baobáxia instalado

Alem das mucuas fixas, tem as mucuas moveis, baseados em pendrive, hd ou um laptops.

Conteúdos estão nas comunidades

Cada comunidade hospeda sua mucua

Os conteúdos ficam hospedados localmente, mas podem ser copiados para outras comunidades

Pela internet, havendo internet..

Se não há internet, uma pessoa pode servir de mensageira, levando um conteúdo de uma comunidade a outra em uma Mucua (que pode ser um laptop, pendrive ou drive externo)

Domínio do território digital. Conteúdos distribuídos pelas comunidades e não num servidor comercial -> autonomia

Como funciona?

Software e Infra-estrutura técnica que vincula os arquivos de cada comunidade numa rede conjunta

Sincronização

Quando uma mucua de uma comunidade se sincroniza com a rede (por internet ou através de outra mucua), ela recebe o "catálogo" de conteúdos atualizado.

Paralelamente, ela envia o catálogo dos conteúdos adicionados localmente para a rede

A mucua pode estar online ou na rede local

É possível ver o catálogo de toda a rede, e conteúdos podem ser solicitados.

Caso haja requisições de conteúdos ativas, durante a sincronização um conteúdo pode ser transferido para o local que o deseja.

Esse transporte pode ser pela internet ou pelo mensageiro, que leva também as listas de desejos das comunidades

Listas de usuários/as de cada comunidade são repassadas para as outras mucuas.

Assim, não é preciso criar novos usuários caso se esteja acessando uma mucua em outro lugar, caso já se tenha login em alguma mucua.

Etiquetas funcionais Possibilidade de expansão das funcionalidades do sistema a partir das etiquetas. 3 camadas: - regras de distribuição e difusão - visualização diferenciada - funcionalidades extra para mídias com determinadas características Exemplos: -> se um conteúdo possui a etiqueta "patrimônio", aquele conteúdo é distribuíudo para mucuas com esse perfil -> se um conteúdo possui a etiqueta "rádio" -> o conteúdo é distribuído para mucuas de rádio, -> a lista de conteúdos gera uma url para uma playlist / streaming de rádio -> se um conteúdo é privado, ele não sai da comunidade e não aparece nas listagem de conteúdos para outras comunidades Uso no Patrimoniamento Está sendo usado e desenhado para atender às necessidades de cadastramento de patrimonio imaterial A lógica distribuída permite que se uma comunidade disser que determinado conteúdo é patrimônio, aquele conteúdo é automaticamente repassado para as mucuas que trabalham com patrimoniamento. Uso escolar Difusão de conteúdos em escala nacional, garantindo o atendimento da Lei 10.639 (difusão de conteúdos de História da África, matriz africana e indígena) Difusão de material de ensino direto e de apoio, como cartilhas, textos, video-aulas, documentários, palestras, músicas etc
